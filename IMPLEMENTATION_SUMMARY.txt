================================================================================
WHAT GETS LOGGED PER PUZZLE
================================================================================

For each of 30 puzzles × 6 paradigms = 180 sessions, the following is logged:

✅ Session Metadata
   - puzzle_id, paradigm, difficulty
   - provider (openai/deepseek/groq)
   - timestamp_start, timestamp_end

✅ Puzzle Metadata (for verification)
   - secret_code: ["red", "blue", "green", "yellow"]
   - available_colors: ["red", "blue", "green", "yellow", "white", "black"]
   - pegs: 4

✅ Task Success Metrics
   - result.success: true/false
   - result.guesses: number of guesses taken
   - result.rounds: number of rounds
   - result.termination_reason: "solution_found" | "max_rounds_reached" | "error"

✅ Token Usage (ACTUAL from API)
   - token_usage.total_input: cumulative input tokens
   - token_usage.total_output: cumulative output tokens
   - token_usage.per_round: [{round, input, output, total}, ...]

✅ Per-Round Details
   - round_data.{N}.guess: the guess made
   - round_data.{N}.feedback: {correct_pegs, correct_positions}
   - round_data.{N}.response_chars: length of LLM response
   - round_data.{N}.model: "gpt-4-turbo" | etc

✅ Inter-Agent Messages (for LLM-Judge evaluation)
   - messages[].timestamp: ISO 8601 timestamp
   - messages[].round: round number
   - messages[].sender: agent name
   - messages[].receiver: agent name
   - messages[].message_type: "strategy_proposal" | "analysis" | etc
   - messages[].content: full message content (dict)

✅ Extracted Constraints
   - constraints_extracted[].round: round number
   - constraints_extracted[].analysis: {impossible, confirmed, locked_positions}
   - constraints_extracted[].timestamp: when extracted

✅ Agent Performance
   - agent_performance.{agent_type}.rounds.{N}: per-round metrics

================================================================================
WHAT ANALYSIS CAN BE DONE
================================================================================

AUTOMATED METRICS (from logs, no extra work):
✅ Success Rate             - % puzzles solved
✅ Avg Guesses              - average guesses for solved puzzles
✅ Failure Rate             - % puzzles not solved
✅ Token Cost               - total tokens used
✅ Tokens/Guess             - efficiency metric
✅ Message Count            - number of inter-agent messages
✅ Message Breakdown        - count by message type

LLM-JUDGE METRICS (Claude API, ~$5-10 total):
⏳ Wasted Comm Rate         - % messages that are non-productive
⏳ Role Adherence           - % messages on-role per agent
⏳ Coordination Score       - 1-5 scale holistic eval

EMBEDDING-BASED METRICS (free, local):
⏳ Convergence Speed        - when strategy becomes stable

import json, glob
from metrics import MetricsAggregator

sessions = {}
for f in glob.glob('output/sessions/*.json'):
    with open(f) as file:
        s = json.load(file)
        p = s['paradigm']
        if p not in sessions: sessions[p] = []
        sessions[p].append(s)

for paradigm, sess in sessions.items():
    agg = MetricsAggregator()
    for s in sess: agg.add_session(s)
    summary = agg.summary()
    print(f'{paradigm}: {summary[\"success_rate\"]:.1f}% success')
"
